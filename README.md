# 命运重开模拟器 2.0

第一阶段基础工程聚焦“高级、克制、有仪式感”的首页体验，采用原生微信小程序工程形态，便于后续平滑扩展命盘页、事件页与结局页。

## 当前阶段范围

- 项目基础骨架
- 统一设计系统
- 首页视觉与交互
- 核心组件抽象
- 后续页面结构预留

## 视觉原则

- 深色沉浸背景与柔金点缀
- 大留白、细边框、玻璃质感与轻粒子
- 命盘核心光效只服务于仪式感，不滥用发光
- 文案短、有命运感，避免营销页口吻

## 目录结构

```text
components/  可复用 UI 组件
constants/   设计系统与常量
data/        首页静态内容数据
pages/       页面入口
styles/      全局 tokens 与基础样式
types/       后续 TypeScript 数据类型预留
utils/       通用工具函数
assets/      后续视觉资产预留
```

## 运行方式

使用微信开发者工具导入当前目录，AppID 可先使用测试号或游客模式预览。
